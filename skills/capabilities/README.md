# `/capabilities` -- shinkoku 対応状況の確認

shinkoku が現在対応している申告パターン、帳票種類、既知の制限事項を一覧表示するスキル。

## このスキルでできること

- 対応ペルソナの一覧表示（Full / Partial / Out の対応レベル付き）
- PDF 帳票の対応状況一覧（確定申告書、青色申告決算書、消費税申告書等）
- e-Tax xtx の対応帳票コード一覧（KOA020、KOA210、SHA010 等）
- 未対応帳票の一覧と代替手段の案内
- e-Tax 電子申告時の添付書類省略ルールの案内
- 対応外ペルソナへの案内テンプレート表示

## 使い方

前提条件は特にない。以下のような場合に使用する:

- 「何ができるの？」「対応している申告は？」と聞かれたとき
- 特定の申告パターンが対応しているか確認したいとき
- 生成可能な帳票の種類を知りたいとき
- 対応外の機能について代替手段を案内したいとき

## 補足

- 対応外のペルソナ（株式分離課税、FX、不動産所得等）については、税理士への相談または国税庁の確定申告書等作成コーナーの利用を案内する
