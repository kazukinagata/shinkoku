# 確定申告書B様式 記載欄（Form B Fields）

確定申告書B（第一表・第二表）の各欄と、対応するデータの対応表。
`calc_income_tax` / `calc_deductions` の出力値を各欄に正しくマッピングするための参照資料。

## 第一表（表面）

### 上段: 基本情報

| 欄 | 項目名 | データソース |
|----|--------|-------------|
| — | 住所 | ユーザー入力 |
| — | 氏名 | ユーザー入力 |
| — | 生年月日 | ユーザー入力 |
| — | 個人番号（マイナンバー） | ユーザー入力 |
| — | 種類（青色 / 分離 / 修正 等） | assess判定結果 |
| — | 税務署名 | ユーザー入力（所轄税務署） |

### 税額の計算

| 欄番号 | 項目名 | 計算方法 |
|--------|--------|----------|
| ㉚ | 課税される所得金額 | ⑫ − ㉙（1,000円未満切り捨て） |
| ㉛ | 上の㉚に対する税額 | 速算表を適用 |
| ㉜ | 配当控除 | 該当する場合 |
| ㉝ | 住宅借入金等特別控除 | housing_loan_credit |
| ㊱ | 差引所得税額 | ㉛ − ㉜ − ㉝（マイナスの場合は0） |
| ㊲ | 復興特別所得税額 | ㊱ × 2.1% |
| ㊳ | 所得税及び復興特別所得税の額 | ㊱ + ㊲ |
| ㊹ | 源泉徴収税額 | 源泉徴収票の税額合計 + 予定納税額 |
| ㊺ | 申告納税額 | ㊳ − ㊹（100円未満切り捨て） |
| ㊻ | 第3期分の税額（納付） | ㊺がプラスの場合 |
| ㊼ | 還付される税金 | ㊺がマイナスの場合（絶対値） |

## 端数処理ルール

| 対象 | ルール |
|------|--------|
| 各所得金額 | 円単位（端数処理なし） |
| 課税所得金額（㉚） | 1,000円未満切り捨て |
| 算出税額（㉛） | 円単位（速算表の結果そのまま） |
| 復興特別所得税（㊲） | 円単位（端数処理なし） |
| 申告納税額（㊺） | 100円未満切り捨て |
| 還付税額（㊼） | 円単位（端数処理なし） |

## 給与所得控除額の速算表（令和7年分）

| 給与等の収入金額 | 給与所得控除額 |
|-----------------|---------------|
| 〜162.5万円 | 65万円（最低保障額） |
| 162.5万超〜180万円 | 収入 × 40% − 10万円 |
| 180万超〜360万円 | 収入 × 30% + 8万円 |
| 360万超〜660万円 | 収入 × 20% + 44万円 |
| 660万超〜850万円 | 収入 × 10% + 110万円 |
| 850万超 | 195万円（上限） |

※ 令和7年分より最低保障額が55万円 → 65万円に引き上げ。
