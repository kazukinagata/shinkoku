# `/document` -- 確定申告書類 PDF 生成

確定申告に必要な PDF 帳票の生成をオーケストレーションするスキル。各帳票の生成はサブエージェントに委任し、メインコンテキストの消費を最小化する。

## このスキルでできること

- 確定申告書 第一表・第二表の PDF 生成
- 青色申告決算書（損益計算書・貸借対照表）の PDF 生成
- 消費税申告書の PDF 生成（課税事業者の場合）
- 医療費控除明細書の PDF 生成（該当者のみ）
- 住宅ローン控除計算明細書の PDF 生成（該当者のみ）
- 第四表（損失申告）の PDF 生成（該当者のみ）
- 全帳票の一括生成
- 生成後の PDF プレビュー（PNG 画像変換による目視確認）

## 使い方

以下のスキルが完了していることが前提条件となる:

1. `/setup` -- 設定ファイルが作成済みであること
2. `/income-tax` -- 所得税の計算が完了していること
3. `/settlement` -- 決算書が作成済みであること

税額計算が完了した後、「書類を作成して」「PDFを生成して」「申告書を出力して」等で呼び出す。

## 補足

- 各帳票の PDF 記入は `/filling-pdf` スキルをサブエージェントに委任して実行する。Claude Code の利用を推奨
- テンプレート PDF は国税庁公式サイトからダウンロードしたものを使用する
- 生成された PDF は `output/` ディレクトリに出力される
