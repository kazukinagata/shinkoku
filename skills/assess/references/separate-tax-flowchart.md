# 分離課税の申告要否 判定フローチャート

assess スキルのステップ2.5で使用する判定フローチャート。

## 株式取引の判定フローチャート

```
Q1. 株式の売却益または配当収入がありますか？
├── No → 【株式の分離課税申告不要】
└── Yes → Q2へ

Q2. 特定口座（源泉徴収あり）のみですか？
├── Yes → Q3へ
└── No → 【確定申告が必要】（第三表を使用）

Q3. 以下のいずれかに該当しますか？
- 複数口座間で損益通算したい
- 前年以前の譲渡損を繰り越して使いたい
- 配当と譲渡損を通算したい
- 配当の課税方式を総合課税にしたい
├── Yes → 【確定申告した方が有利】（第三表を使用）
└── No → 【確定申告不要】（源泉徴収で課税完了）
```

## FX取引の判定フローチャート

```
Q1. FX（外国為替証拠金取引）の損益がありますか？
├── No → 【FXの分離課税申告不要】
└── Yes → Q2へ

Q2. 利益が出ていますか？
├── Yes → 【確定申告が必要】（先物取引に係る雑所得等、分離課税）
└── No → Q3へ

Q3. 翌年以降に損失を繰り越したいですか？
├── Yes → 【確定申告が必要】（損失の繰越控除の適用）
└── No → 【確定申告不要】
```

## 仮想通貨の判定フローチャート

```
Q1. 暗号資産の売却・交換・決済による所得がありますか？
├── No → 【仮想通貨の申告不要】
└── Yes → Q2へ

Q2. 年間の所得金額（利益 - 経費）はいくらですか？
├── 20万円以下（給与所得者の場合）→ 【所得税の申告不要】（ただし住民税は申告必要）
└── 20万円超 → 【確定申告が必要】（雑所得として総合課税）
```
