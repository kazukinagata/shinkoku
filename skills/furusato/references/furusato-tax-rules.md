# ふるさと納税の税法リファレンス

## 法令根拠

- **所得税法第78条**（寄附金控除）
- **地方税法第37条の2**（寄附金税額控除・ふるさと納税特例控除）

## 控除の仕組み

### 所得控除（所得税）

- 控除額 = 寄附合計額 - 2,000円
- 上限: 総所得金額の40%
- 寄附金控除は「所得控除」（税額控除ではない）

### 住民税からの控除

1. **基本分**: (寄附合計 - 2,000) x 10%
2. **特例分**: (寄附合計 - 2,000) x (100% - 10% - 所得税率 x 1.021)
   - 上限: 住民税所得割額の20%

### 控除上限額の推定式

```
上限 = 住民税所得割額 x 20% / (100% - 所得税率 x 1.021 - 10%) + 2,000
```

- 住民税所得割額 = 課税所得 x 10%
- 所得税率は課税所得に応じた累進税率

### 推定が外れるケース

- 調整控除（人的控除の差）を未考慮
- ふるさと納税以外の寄附金控除がある場合
- 住宅ローン控除で住民税が減額されている場合

## ワンストップ特例制度

### 要件
- 確定申告が**不要**な給与所得者等
- 寄附先が**5自治体以下**

### 確定申告時の取り扱い
- 確定申告を行うとワンストップ特例は**全て無効化**される
- 全寄附分を確定申告で寄附金控除として申告する必要がある
- 副業で確定申告する本プラグインのユーザーは必ず確定申告が必要

## 令和7年分の留意点

- 基本的な計算ルールに変更なし
- 基礎控除の引き上げ（58万→95万等）により控除上限が変動する可能性あり
