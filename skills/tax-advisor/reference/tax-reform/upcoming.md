# 翌年以降の予定改正（Upcoming Tax Reforms）

令和7年分（2025年課税年度）時点で予定・議論されている将来の税制改正。

**注意**: この情報は将来の見通しであり、確定したものではない。
実際の改正内容は各年度の税制改正大綱・法改正により確定する。

## 令和8年分（2026年）以降に影響する予定事項

### 1. インボイス2割特例の終了

| 項目 | 内容 |
|------|------|
| 経過措置の終了 | 令和8年9月30日を含む課税期間が最終 |
| 令和9年分以降 | 2割特例は適用不可（延長の議論あり） |

**影響**: インボイス登録により課税事業者になった小規模事業者は、令和9年分から
簡易課税または本則課税で申告する必要がある。

**対策**:
- 簡易課税の届出を検討（令和8年12月31日までに届出）
- 本則課税の場合は帳簿の税率区分記載を整備
- 免税事業者に戻ることの検討（基準期間の売上1,000万円以下の場合）

### 2. 免税事業者からの仕入控除割合の段階的引下げ

| 期間 | 控除割合 |
|------|---------|
| 〜令和8年9月30日 | 80%（現在） |
| 令和8年10月1日〜令和11年9月30日 | **50%** |
| 令和11年10月1日〜 | **0%** |

**影響**: 免税事業者との取引がある課税事業者の仕入税額控除額が段階的に減少。
**対策**: 取引先のインボイス登録状況の確認、価格交渉の検討。

### 3. 復興特別所得税の見直し

| 項目 | 内容 |
|------|------|
| 現行 | 令和19年（2037年）まで、所得税額の2.1% |
| 見直しの議論 | 税率を引き下げて課税期間を延長する案あり |

防衛費財源の確保との関連で、復興特別所得税の一部を防衛費に転用し、
代わりに課税期間を延長する案が議論されている。

### 4. 少額減価償却資産特例の期限

| 項目 | 内容 |
|------|------|
| 現行の期限 | 令和8年3月31日 |
| 見通し | 過去に複数回延長されており、今後も延長される可能性が高い |

### 5. セルフメディケーション税制の期限

| 項目 | 内容 |
|------|------|
| 現行の期限 | 令和8年12月31日 |
| 見通し | 延長の有無は令和8年度税制改正大綱で決定 |

## 中長期的な税制の方向性（議論段階）

### 給与所得控除の見直し

- 上限額（195万円）の引下げ議論
- フリーランス等との公平性の観点から、給与所得控除の水準の見直し議論あり

### 退職所得課税の見直し

- 勤続年数20年超の退職所得控除の上乗せ（年70万→年40万円への引下げ）議論
- 短期間の転職が不利にならない制度への見直し

### 金融所得課税の強化

- 金融所得に対する課税強化（最低税率の導入等）の議論
- 1億円の壁（所得1億円超で実効税率が下がる問題）への対応

### 社会保険料の「壁」対策

- 106万円・130万円の壁の見直し議論
- 被扶養者の収入要件の段階的な引上げ
- 短時間労働者の社会保険適用拡大の継続

### デジタル化・電子帳簿保存法

- 電子帳簿保存法の要件の段階的な厳格化
- 電子取引データの電子保存義務（すでに義務化、猶予措置の終了）
- e-Taxの利用促進策

## 予定改正への対応方針

### 確実に対応が必要なもの

1. **インボイス2割特例の終了準備**（令和8年度中に対応）
2. **免税事業者仕入控除50%→0%への段階的対応**
3. **電子帳簿保存法への完全対応**

### 動向を注視すべきもの

1. 基礎控除・給与所得控除のさらなる見直し
2. 退職所得課税の見直し
3. 金融所得課税の強化
4. 社会保険料の壁の見直し

### shinkoku プロジェクトへの影響

- 毎年の税制改正大綱（12月閣議決定）を反映して reference ファイルを更新
- tax-reform/ ディレクトリに新しい年度ファイル（例: 2026.md）を追加
- 税額計算ロジック（tax_calc.py）のパラメータを年度ごとに管理

## 免責事項

実際の改正内容は将来の税制改正法により確定する。
