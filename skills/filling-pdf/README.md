# `/filling-pdf` -- PDF 帳票記入ワークフロー

国税庁公式 PDF テンプレートに計算結果を書き込み、提出用 PDF を生成するサブエージェント用ワークフロー。

## このスキルでできること

- 国税庁テンプレート PDF への税額・氏名・住所等の自動記入
- 帳票種類に応じた座標定義に基づく正確な位置への描画
- テンプレート PDF が未取得の場合の自動ダウンロード
- 生成後の PDF プレビュー（PNG 変換）と位置ずれの検証

### 対応帳票

- 確定申告書 第一表・第二表
- 青色申告決算書 損益計算書（P1-P3）・貸借対照表
- 第四表（損失申告）
- 消費税申告書
- 医療費控除の明細書
- 住宅ローン控除計算明細書

## 使い方

このスキルはユーザーが直接呼び出すものではない。`/document` スキルが各帳票の生成時にサブエージェントへ委任する際に使用される。

サブエージェントには以下の情報が渡される:

- 帳票種類とテンプレート PDF のパス
- 入力データ（計算結果 + プロフィール情報）
- 出力先パス

## 補足

- Claude Code のサブエージェント機能を前提としたワークフロー。Claude Code の利用を推奨
- 座標定義は `src/shinkoku/tools/pdf_coordinates.py` に集約されている
- フィールドとデータソースの対応は `references/` ディレクトリ内の各帳票定義ファイルを参照
