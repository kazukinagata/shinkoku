# 提出前チェックリスト詳細

submit スキルのステップ1で使用する全チェック項目。

## 1-1. 所得税の確定申告書チェック

### 第一表の確認事項

```
[] 住所・氏名・生年月日・マイナンバーが正しいか（profile_get_taxpayer で確認）
[] 種類欄のチェック（青色 / 分離 / 修正 等）
[] 収入金額等（ア〜サ）の各欄が正しいか
  - 給与: 源泉徴収票の「支払金額」と一致するか
  - 事業: 青色申告決算書の売上金額と一致するか
[] 所得金額等（1〜12）が正しいか
  - 給与所得: 給与所得控除後の金額（源泉徴収票と一致）
  - 事業所得: 青色申告決算書の所得金額と一致するか
[] 所得から差し引かれる金額（所得控除）
  - 社会保険料控除: 源泉徴収票の金額 + 追加分
  - 生命保険料控除: 控除証明書の金額と照合
  - 基礎控除: 令和7年分の段階表に基づく正しい金額か
  - その他の控除が漏れていないか
[] 税額の計算
  - 課税所得金額: 1,000円未満切り捨てされているか
  - 所得税額: 速算表の適用が正しいか
  - 税額控除（住宅ローン控除等）が反映されているか
  - 復興特別所得税（2.1%）が加算されているか
  - 源泉徴収税額が正しく控除されているか
  - 申告納税額: 100円未満切り捨てされているか
[] 還付される税金の受取場所（口座情報）
[] 住民税に関する事項
  - 給与・公的年金等に係る所得以外の住民税の徴収方法
    （普通徴収 or 特別徴収）
```

### 第二表の確認事項

```
[] 所得の内訳（所得税及び復興特別所得税の源泉徴収税額）
  - 各支払先の源泉徴収税額の合計が第一表の(49)と一致するか
[] 社会保険料控除の内訳
[] 生命保険料控除の内訳（新旧区分が正しいか）
[] 配偶者や親族に関する事項（該当者のみ）
[] 住民税に関する事項
```

## 1-2. 青色申告決算書のチェック（該当者のみ）

```
[] 損益計算書
  - 売上金額の合計が実績と一致するか
  - 経費の各科目が帳簿と一致するか
  - 青色申告特別控除額（65万円 / 10万円）が正しいか
  - 65万円控除の3要件:
    (1) 複式簿記で記帳しているか
    (2) e-Taxで提出する or 電子帳簿保存しているか
    (3) 期限内（3月15日まで）に申告するか
    ※ 1つでも欠けると10万円控除に

[] 貸借対照表
  - 資産合計 = 負債・資本合計 であるか
  - 期首残高が前年の期末残高と一致するか（2年目以降）
  - 事業主貸・事業主借の金額が合理的か
```

## 1-3. 消費税申告書のチェック（該当者のみ）

```
[] 課税期間が令和7年1月1日〜12月31日であるか
[] 申告区分（2割特例/簡易課税/本則課税）が正しいか
[] 課税売上高が帳簿の合計と一致するか
[] 納付税額の計算が正しいか
  - 2割特例: 課税標準額に対する消費税額 x 20%
  - 簡易課税: みなし仕入率の適用が正しいか
  - 本則課税: 課税仕入の集計が正しいか
[] 中間納付がある場合、控除されているか
```

## 1-3.5. 第三表（分離課税）のチェック（該当者のみ）

株式取引や FX 取引で分離課税の申告がある場合:

```
[] 第三表（分離課税用）が生成されているか
[] 株式等の譲渡所得が正しく記載されているか
[] 配当所得の課税方式（総合/分離/申告不要）が意図通りか
[] 先物取引に係る雑所得等（FX）が正しく記載されているか
[] 繰越損失がある場合、損失の繰越控除が正しく適用されているか
[] 源泉徴収済み税額が正しく控除されているか
[] 株式とFXの損益が別プールで計算されているか（損益通算不可）
```

## 1-4. 添付書類のチェック

```
[] 源泉徴収票（e-Taxの場合は添付省略可）
[] 各種控除証明書（e-Taxの場合は添付省略可の場合あり）
[] 医療費控除の明細書（該当者のみ）
[] 寄附金受領証明書（ふるさと納税、該当者のみ）
[] 住宅ローン控除関連書類（初年度の場合）
  - 住宅借入金等特別控除額の計算明細書
  - 年末残高等証明書
  - 登記事項証明書
  - 売買契約書等の写し
[] 特定口座年間取引報告書（株式取引がある場合）
[] 先物取引に係る年間損益報告書（FX取引がある場合）
[] 暗号資産の計算書（仮想通貨取引がある場合）
[] セルフメディケーション税制の証明書（該当者のみ）
  - OTC医薬品の購入レシート
  - 特定健康診査等の結果通知表
```
